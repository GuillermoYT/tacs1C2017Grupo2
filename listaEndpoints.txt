Como usuario quiero poder crear una cuenta con user y pass: [LISTO]

	POST : http://localhost:8080/usuarios?name={usuario}&pass={pass}

Como usuario quiero hacer log in en la página: [LISTO]

	POST : http://localhost:8080/user/login?name={usuario}&pass={pass}

Check de login: 

	PUT : http://localhost:8080/user/auth

Como usuario quiero hacer log out en la página: [LISTO]

	PUT : http://localhost:8080/user/logout?id={id}
	
Como usuario quiero buscar por nombre (películas y actores):[LISTO]

	GET : http://localhost:8080/usuarios/buscar/{pelicula}/{actor}

    Buscar una pelicula:[LISTO]
	
		GET : http://localhost:8080/usuarios/buscar/pelicula/{pelicula}
	
	Buscar un Actor:[LISTO]

		GET : http://localhost:8080/usuarios/buscar/actor/{actor}
		
Como usuario quiero crear una lista de películas con un nombre: [LISTO]

	POST : http://localhost:8080/movielist/{usuario}

    Agregar películas a la lista: [LISTO]

		PUT : http://localhost:8080/movielist/{usuario}
  
	Eliminar películas de la lista: [LISTO]

		DELETE : http://localhost:8080/movielist/{usuario}/{movie}
    
	Consultar la lista: [LISTO]

		GET : http://localhost:8080/movielist/{usuario}

Como usuario quiero ver la intersección de películas entre 2 listas: [LISTO]

	POST : http://localhost:8080/movielist/{usuario}/intersection
	
	JSON : 
	{"peli1":[{"id":1,"nombre":"The Matrix","nacionalidad":"USA","anioEstreno":1999},{"id":0,"nombre":"Noé","nacionalidad":"USA","anioEstreno":2014}],
	"peli2":[{"id":1,"nombre":"The Matrix","nacionalidad":"USA","anioEstreno":1999}]}
	
Como usuario quiero marcar un actor como favorito: [LISTO] 

	PUT : http://localhost:8080/actores/{usuario}/favorito/{actor}

Como usuario quiero marcar un actor como favorito: [LISTO]

	DELETE : http://localhost:8080/actores/{usuario}/favorito/{actor}

Como usuario quiero ver quienes son mis actores favoritos:	[LISTO]

	GET : http://localhost:8080/actores/{usuario}/actoresFavoritos

Como usuario quiero un ranking de actores que se repitan en las películas de una lista: [LISTO]

	GET : http://localhost:8080/movielist/{usuario}/ranking/actoresFavoritosDePeliculas
		
Como administrador quiero ver el ranking de actores favoritos de todos mis usuarios (los actores que han sido más veces elegidos como favoritos)): [LISTO]

	GET : http://localhost:8080/movielist/{administrador}/ranking/{idList}

Como usuario quiero poder ver el detalle de una película: [LISTO]

	GET : http://localhost:8080/movielist/{usuario}/{pelicula}

Como usuario quiero poder ver el detalle de un actor: [LISTO]

	GET : http://localhost:8080/actores/{usuario}/{actor}

Como administrador quiero poder ver los siguientes datos de un usuario: [LISTO]

	GET : http://localhost:8080/usuarios/{usuario}

Como administrador quiero seleccionar 2 listas de usuarios diferentes y mostrar la intersección de películas: [LISTO]

	GET : http://localhost:8080/movielist/compare?user1={usuario1}&user2={usuario2}

Como usuario quiero ver si hay películas donde esté más de uno de mis actores favoritos: [LISTO]

	GET : http://localhost:8080/movielist/{usuario}/actoresFavoritos
